# MOST-style GitHub Pages Conference Website

This is a simple static HTML/CSS conference website inspired by the layout in the provided screenshot.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** root
5. Save.
6. GitHub will give you a URL like:
   `https://your-username.github.io/repository-name/`

## To use a custom domain

1. Buy a domain, for example `yourconference.org`.
2. In GitHub: **Settings → Pages → Custom domain**.
3. Add your domain.
4. In your domain provider DNS settings, point it to GitHub Pages.

## Notes

- Edit `index.html` for the homepage.
- Edit `assets/css/style.css` for layout, colors, and spacing.
- Replace placeholder logos/text with your own conference information.
- This version uses only HTML/CSS, so it works on GitHub Pages.
